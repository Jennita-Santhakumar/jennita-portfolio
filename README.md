# Jennita S — Portfolio Website

A clean, modern, dark-themed personal portfolio built with vanilla HTML, CSS, and JavaScript.
No frameworks, no build tools — just open `index.html` in any browser.

## 📁 Files
- `index.html` — Main portfolio page
- `style.css`  — All styling (CSS variables, responsive layout, animations)
- `script.js`  — Scroll reveal, active nav, cursor glow

## 🚀 How to Deploy

### Option 1: GitHub Pages (Free)
1. Create a new GitHub repo (e.g. `jennita-portfolio`)
2. Upload all 3 files to the repo
3. Go to Settings → Pages → Source: main branch → Save
4. Your site will be live at `https://yourusername.github.io/jennita-portfolio`

### Option 2: Netlify (Free, Drag & Drop)
1. Go to https://netlify.com
2. Drag and drop this folder into the deploy zone
3. Done — live URL in seconds

### Option 3: Vercel (Free)
1. Go to https://vercel.com
2. Import your GitHub repo or drag & drop
3. Deploy

### Option 4: Local Preview
Just open `index.html` in your browser — no server needed.

## ✏️ Customization Tips
- Update LinkedIn/GitHub URLs in `index.html` (search for `href="#"`)
- Colors are in `:root` in `style.css` — change `--accent` to match your brand
- Add your photo by replacing the hero section background
- Add a `favicon.ico` for the browser tab icon

## 📱 Responsive
Works on mobile, tablet, and desktop.
